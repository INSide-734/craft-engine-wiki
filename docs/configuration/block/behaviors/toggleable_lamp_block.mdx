| Property Name | Property Type | Required |
|---------------|---------------|----------|
| lit           | boolean       | yes      |
| powered       | boolean       | yes      |

**Toggleable Lamp Block** allow blocks to cycle between `lit` states when activated by redstone signals.

```yaml
blocks:
  default:copper_coil:
    behavior:
      type: toggleable_lamp_block
      can-open-with-hand: false # When set to true, all redstone features will be disabled
```
