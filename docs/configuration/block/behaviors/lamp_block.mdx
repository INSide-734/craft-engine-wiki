| Property Name | Property Type | Required |
|---------------|---------------|----------|
| lit           | boolean       | yes      |
| powered       | boolean       | yes      |

**Lamp Block** allows the block to turn into `lit` state on redstone powered. 

```yaml
blocks:
  default:copper_coil:
    behavior:
      type: lamp_block
      can-open-with-hand: false # When set to true, all redstone features will be disabled
      redstone-toggle-mode: false
```
