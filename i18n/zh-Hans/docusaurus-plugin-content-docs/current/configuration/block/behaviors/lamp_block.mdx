---
title: 💡 灯方块
id: lamp_block
---

| 属性名称    | 属性类型    | 是否必需 |
|---------|---------|------|
| lit     | boolean | 是    |
| powered | boolean | 是    |

**灯方块**允许方块在红石信号激活时切换至 `lit` 点亮状态。

```yaml
blocks:
  default:copper_coil:
    behavior:
      type: lamp_block
      can-open-with-hand: false # 当设置成 true 时，全部的红石特性将会被禁用
      redstone-toggle-mode: false
```
